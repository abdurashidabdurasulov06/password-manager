# 🔐 Team Password Manager

A secure web application for storing and managing shared company credentials.

## 🚀 Features

- Master password protection
- AES-encrypted password storage
- Add / edit credentials
- Password generator
- Password strength indicator
- Copy to clipboard
- Password reuse detection
- Data export
- Audit log (extra feature)

## 🛠 Tech Stack

- Node.js (Express)
- SQLite
- Vanilla JavaScript
- HTML/CSS

## ▶️ How to Run

1. Clone repo:
git clone https://github.com/abdurashidabdurasulov06/password-manager.git

2. Open project:
cd password-manager

3. Install dependencies:
npm install

4. Run server:
node server/server.js

5. Open browser:
http://localhost:3000

## 🌍 Deployment


Team Vault is a secure password manager designed for companies that share credentials across teams and tools. It eliminates insecure practices like sending passwords through chats by storing all data in an encrypted vault protected by a master password. The app includes password generation, reuse detection, and audit logging for better security. In version 2, we would introduce role-based access control, team permissions, and cloud synchronization.

If I had 3 more days, I would implement role-based access control so different team members can have different permissions (view-only, edit, admin). I would also add two-factor authentication (2FA) for unlocking the vault to increase security. Finally, I would improve the UI using a modern design system like Tailwind to make the product feel more trustworthy and professional.
